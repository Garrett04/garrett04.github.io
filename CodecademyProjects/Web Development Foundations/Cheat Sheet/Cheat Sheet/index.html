HTML Cheat Sheet

HTML Tags
Tag Name Description
<nav> Navigation

Consists a set of navigational links. Mostly used to navigate different sections in a document.

<section> Section

Defines a section in a document.

<article> Article

Represents a section of content that forms an independent part of a document or site. Example: Magazine, newspaper article, etc.

<blockquote> Blockquote

A section[within a document] that is quoted from another source.

<span>
&
<div>		 Span and Division

Span is used to mark up a part of text inside a paragraph while Division tag is used to define part of a document.

<img> Image

To add an image using the src attribute.

<figure> Figure

Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.

HTML Attributes
Attribute Name Description
id ID

Specifies a unique id for an HTML element. The value of an id attribute must be unique.

class Class

Specifies a class for an HTML element. Multiple HTML elements can share the same class unlike id.

href Hypertext Reference

Commonly used in anchor tags (<a>). Indicates the destination of the hyperlink.

src Source

Specifies the location (URL) of the external resource.
